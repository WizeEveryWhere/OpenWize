
Simulation application
======================

- This application is provide as a simple demonstrator of OpenWize stack targeting 


Requirement
-----------

Linux RT




UART4 (due to wake-up pin availability on PA0, UART4 RX/TX pins are swapped) 
   - This UART is mapped onto : 
      - RX : PA0 (CN7-28)
      - TX : PA1 (CN7-30).
